# 电商系统SQL实战项目
## 项目介绍
本项目基于MySQL设计电商核心业务的数据库结构，包含用户、商品、订单、购物车等核心表，覆盖电商常见业务场景的SQL查询、存储过程开发，旨在展示数据库设计和SQL实战能力。

## 技术栈
- 数据库：MySQL 8.0
- 核心技术：表结构设计、多表关联查询、聚合统计、存储过程、事务

## 项目结构
- `docs/ER图.md`：数据库ER图（表关系设计）
- `sql/01_create.sql`：数据库&表结构创建
- `sql/02_data.sql`：测试数据插入
- `sql/03_business.sql`：电商常用业务查询（销量统计、用户消费分析等）
- `sql/04_procedures.sql`：实用存储过程（订单编号生成、库存扣减等）

## 使用方法
1. 执行`01_create.sql`创建数据库和表结构
2. 执行`02_data.sql`插入测试数据
3. 执行`03_business.sql`查看业务查询效果
4. 调用`04_procedures.sql`中的存储过程体验进阶功能

## 核心亮点
- 表结构设计符合第三范式，兼顾性能（合理索引）
- 业务查询覆盖电商运营核心场景（销量统计、用户画像）
- 存储过程包含事务处理，保证数据一致性
